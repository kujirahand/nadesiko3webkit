# nadesiko3webkit

nadesiko3 for webkit

# コンパイルに必要なライブラリ

 - Go言語

## コンパイルの方法

必要なモジュールを取得する。

```bash
go get -u
```

## Windowsの場合

WindowsではWebView2のランタイム「[こちら](https://developer.microsoft.com/en-us/microsoft-edge/webview2/)」が必要です。

```bash
batch¥build-win.bat
```

## macOSの場合

```bash
batch/build-mac.sh
```
